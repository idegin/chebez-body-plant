# 1. Design Philosophy

## Design Direction

The Bodyplant website should feel like the intersection of **modern wellness**, **premium food branding**, and **scientific credibility**.

Unlike many Nigerian food websites that feel corporate or outdated, Bodyplant should immediately communicate:

* Natural
* Healthy
* Fresh
* Premium
* Trusted
* Modern
* Accessible

The visual language should resemble the quality of premium international health-food brands while remaining authentic to an African audience.

### Keywords

* Organic
* Human
* Fresh
* Clean
* Bright
* Minimal
* Calm
* Trustworthy
* Nutritious
* Premium

* Nature-inspired

# 2. Current Industry Design Trends (2026)

Across leading health food, functional nutrition, and premium beverage brands, the strongest UI trends include:

* Large editorial-style hero photography with authentic lifestyle scenes.
* Soft organic shapes instead of rigid geometric layouts.
* Spacious layouts with generous white space.
* Rounded cards and containers inspired by natural forms.
* Earth-tone color palettes with vibrant botanical accents.
* Minimal navigation emphasizing products and brand values.
* Large typography paired with concise messaging.
* Subtle glassmorphism used sparingly for floating UI elements.
* Scroll-triggered animations that feel smooth rather than distracting.
* Product storytelling supported by icons, illustrations, and ingredient imagery.
* High-quality macro photography showcasing texture and ingredients.
* Mobile-first layouts with touch-friendly controls.
* Accessibility-first design, including high contrast and readable typography.

The goal is to create an experience that feels premium and trustworthy without becoming overly clinical.

# 15. Motion Design

Motion should be subtle and purposeful.

Recommended animations:

* Fade-up on scroll.
* Staggered card reveals.
* Gentle image zoom (1.03×) on hover.
* Button elevation transitions.
* Counter animations for statistics.
* Smooth accordion expansion.
* Horizontal product carousel transitions.
* Parallax movement for ingredient imagery in the hero section.

Avoid excessive motion that distracts from the content.

# 19. Accessibility

* Meet WCAG 2.2 AA standards.
* Minimum body text size of 16px.
* Contrast ratio of at least 4.5:1.
* Visible keyboard focus indicators.
* Descriptive alt text for all imagery.
* Touch targets of at least 44×44px.
* Motion reduction support for users who prefer reduced motion.

---

# 20. Responsive Design

Design should be mobile-first.

Breakpoints:

* Mobile: 360–767px
* Tablet: 768–1023px
* Laptop: 1024–1439px
* Desktop: 1440px+
* Large Desktop: 1920px+

On smaller screens:

* Collapse navigation into a slide-over menu.
* Stack content vertically.
* Convert multi-column grids into single-column layouts where appropriate.
* Keep CTAs prominent and easy to reach with the thumb.

# 21. Visual Storytelling

Every page should balance four visual themes:

1. **Nature** — ingredients, leaves, dates, earth tones, organic forms.
2. **Health** — families, nutritious meals, active lifestyles.
3. **Trust** — certifications, quality assurance, manufacturing standards.
4. **Business** — wholesale partnerships, distribution, logistics, retailer success.

The result should be a website that feels premium, warm, modern, and credible—positioning Bodyplant Nature Resources as a trusted leader in plant-based nutrition and wholesale distribution.

## 22. Technology & Stack

| Concern | Decision |
| --- | --- |
| Framework | Next.js 16 (App Router, Turbopack, React 19) — read `node_modules/next/dist/docs/` before touching framework APIs |
| Styling | Tailwind CSS v4 — tokens defined in `app/globals.css` via `@theme` (no `tailwind.config.js`) |
| Type / Fonts | `next/font/google` self-hosting — **Manrope** (`--font-manrope`, 400–800) + **DM Serif Display** (`--font-dm-serif`, 400) |
| Motion | **Framer Motion** (`motion` package, `motion/react`) for entrance/scroll/gesture; CSS keyframes for ambient loops |
| Icons | Inline SVG (Phosphor-style rounded geometry), 1.9–2.2 stroke, `currentColor` |
| Images | `next/image` with static import (auto width/height + blur placeholder) |
| Rendering | Server Components by default; `"use client"` only for motion/interactive leaves |

## 23. Design Tokens (canonical names)

These are the literal Tailwind v4 token names. **Always use the token, never a raw hex.**

**Color** — `earth-orange` `warm-clay` `natural-green` `deep-green` · surfaces `cream` `pure-white` `warm-gray` `light-sage` · ink `charcoal` `muted` `hairline` · accents `date-fruit` `golden-honey` `leaf-green` `soft-olive`. Semantic aliases: `background`→cream, `foreground`→charcoal, `primary`→deep-green, `accent`→earth-orange.

**Radius** (`var(--radius-*)`): `button` 16 · `input` 16 · `card` 24 · `image` 28 · `hero` 40 · `pill` 999.

**Elevation**: `--shadow-card` `0 12px 40px rgba(0,0,0,.08)` · `--shadow-card-hover` green-tinted lift · `--shadow-float` floating chips · `--shadow-cta` green CTA glow.

**Type scale** (fluid `clamp`): `--text-hero` `--text-h1…h5` `--text-body-lg` `--text-body` `--text-small` `--text-caption`. Headings → serif; body/UI → Manrope. Measure capped ~60–75ch.

**Layout**: `--container-shell` 1320px · `--container-content` 1200px · use the `.container-shell` utility (auto-centered, fluid gutter).

## 24. Motion System

Purposeful, calm, never distracting. Honour `prefers-reduced-motion` everywhere (a global reset kills animation, and all Framer components take a reduced path).

| Token | Curve | Use |
| --- | --- | --- |
| `--ease-organic` | `cubic-bezier(.22,1,.36,1)` | reveals, hovers, layout |
| `--ease-out-soft` | `cubic-bezier(.16,1,.3,1)` | fades, opacity |
| `--ease-spring` | `cubic-bezier(.34,1.56,.64,1)` | CTA arrows, tactile pop |

- **Durations:** UI/micro 150–300ms; entrance 600–900ms; ambient loops 5–16s.
- **Primitives:** `Reveal` (fade-up on scroll), `Stagger`/`staggerItem` (cascades, ~90ms step), floating chips, `animate-blob`, `animate-marquee`, `animate-pulse-ring`, hover lift `translateY(-4…-6px)`, image zoom 1.03×.
- **Reveal on scroll:** `whileInView` + `viewport={{ once:true, margin:"-80px" }}`. Above-the-fold hero animates on mount (not on scroll).

## 25. Component Specs

- **Button** (`components/ui/button.tsx`) — `primary` deep-green → hover earth-orange, `--shadow-cta`, lift on hover; `secondary` white + green ring → hover light-sage; `text` with spring arrow. Radius 16, sizes `md` h-12 / `lg` h-14. All states: hover, `focus-visible` (3px deep-green ring), active (tap 0.97), disabled.
- **Card** — white surface, radius 24, `--shadow-card`, generous padding, hover lift + `--shadow-card-hover`, icon tile flips light-sage→deep-green on hover.
- **Header** — sticky pill; transparent over hero → cream/80 blur + float shadow after 24px scroll; desktop center-nav w/ animated underline; mobile full-height slide-over (staggered serif links, body scroll-lock, backdrop).
- **Section rhythm** — alternate White → Cream → Light Sage → Warm Gray. Section padding 96–160px. Organic SVG wave dividers between contrasting bands.

## 26. Acceptance Criteria (Definition of Done)

- [ ] WCAG 2.2 AA: body ≥16px, contrast ≥4.5:1 (3:1 large/UI), visible focus, targets ≥44px, meaningful `alt`, semantic landmarks + ordered headings, skip link.
- [ ] Responsive 360 → 1920px; no horizontal scroll or CLS; fluid type via `clamp`.
- [ ] `prefers-reduced-motion` fully honoured.
- [ ] Every interactive element has hover / focus-visible / active / disabled; every async action has loading + empty + success states.
- [ ] Zero raw hex in components — tokens only. `pnpm build` type-clean.
- [ ] LCP image prioritised; fonts `display:swap`; transform/opacity-only animation.

## 27. Anti-Patterns (no "AI slop")

Avoid: purple/blue gradients, Inter/Roboto/system defaults, evenly-weighted centered stacks, flat rainbow palettes, generic stock imagery, drop-shadow overload, decorative motion without purpose, emoji as iconography. Favour: one dominant colour (deep-green) + disciplined earth-orange accent, editorial serif display, asymmetry and generous negative space, earned depth (grain, soft mesh, hairlines), and botanical organic forms true to the brand.
